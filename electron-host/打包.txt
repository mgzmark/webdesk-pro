设置打包镜像（针对国内网络环境）

Cmd：
set ELECTRON_BUILDER_BINARIES_MIRROR=https://npmmirror.com/mirrors/electron-builder-binaries/

Powershell
$env:ELECTRON_BUILDER_BINARIES_MIRROR="https://npmmirror.com/mirrors/electron-builder-binaries/"

执行打包命令：npm run build

获取你的 .exe 文件
打包过程可能需要 1~3 分钟，期间终端会输出打包进度。
当看到 Building NSIS installer 并且最终显示 Done 时，说明打包成功了！
打包完成后，在你的 electron-host 文件夹下会自动生成一个名为 dist 的新文件夹。
打开 dist 文件夹，你会在里面找到：
WebDesk Host Setup 1.0.0.exe （这是一个安装程序，双击可以像普通软件一样安装到电脑上）
win-unpacked 文件夹（这是一个免安装的绿色版，里面有一个 WebDesk Host.exe 可以直接双击运行）
你可以把这个 .exe 发给任何人，他们不需要安装 Node.js 也不需要懂代码，双击就能运行被控端了！如果在打包过程中遇到任何红色的报错，请随时把报错信息发给我。
